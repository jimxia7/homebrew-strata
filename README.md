# Homebrew Cask for Strata

[Strata](https://github.com/arkottke/strata) is a seismic site response
analysis application that performs equivalent linear analysis.

## Install

```
brew install --cask arkottke/strata/strata
```

Or, as two steps:

```
brew tap arkottke/strata
brew install --cask strata
```

## Upgrade

```
brew upgrade --cask strata
```

## Uninstall

```
brew uninstall --cask strata
```
